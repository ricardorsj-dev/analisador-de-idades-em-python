# 📊 Analisador de Idades em Python

Projeto desenvolvido em Python com o objetivo de coletar dados de pessoas e realizar análises básicas como:

- Média de idade
- Pessoa mais velha
- Pessoa mais nova

## 🚀 Tecnologias utilizadas

- Python 3

## 📌 Funcionalidades

✔ Entrada de dados com validação  
✔ Cálculo automático da média  
✔ Identificação da pessoa mais velha  
✔ Identificação da pessoa mais nova  

## 💻 Como executar

```bash
git clone https://github.com/ricardors-dev/analisador-de-idades-em-python.git
cd analisador-idades
python main.py
